Import Requirements

Fingerprint folder:

  • Upload a folder containing BMP images
  • Filename format must be:
  • externalId__sex_hand_finger.bmp
  • Example:
  • 1__m_left_thumb_finger.bmp
  • 25__f_right_index_finger.bmp

Allowed values:

  • sex: m or f
  • hand: left or right
  • finger: thumb, index, middle, ring, little

Subjects CSV format:

external_id, first_name, last_name, age, address, city, country, phone_number

Important:

  • external_id must match fingerprint filename ID
  • CSV must be UTF-8 encoded
  • No missing columns allowed
